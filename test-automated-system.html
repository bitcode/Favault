📋 Testing Instructions for Brave Browser

Step 1: Load the FaVault extension in Brave
1. Open Brave browser 2. Go to brave://extensions/ 3. Enable "Developer mode" (top right) 4. Click "Load unpacked" 5. Select the dist/production folder from your FaVault project
Step 2: Open a new tab to trigger the extension
1. Open a new tab (Ctrl+T) 2. The FaVault extension should load automatically 3. Wait for bookmarks to load
Step 3: Open Developer Console
1. Press F12 or Ctrl+Shift+I 2. Go to the "Console" tab 3. Look for initialization messages starting with "🔧"
Step 4: Test the automated functions
// Run these commands in the console: runAllTests() // Comprehensive test suite testMove(4, 2) // Test individual move showState() // Show current state getTestResults() // View test results

🔧 Troubleshooting

❌ If functions are not available:

  • Check that the extension loaded properly (no errors in console)
  • Verify the extension has bookmark permissions
  • Try refreshing the new tab page
  • Check for any TypeScript/build errors

❌ If tests fail:

  • Ensure you have bookmarks in your Brave browser
  • Check that the Chrome bookmark API is accessible
  • Verify folder containers are present in the DOM
  • Try running individual test functions

✅ Success Indicators:

  • Functions available: typeof runAllTests === 'function'
  • No manual copy-paste needed
  • Concise, informative console output
  • Test results automatically exported
  • Real bookmark operations working
